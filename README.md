# 🚀 Udemig Dijital Web Projesi

Modern ve şık bir web arayüzü. Bu proje HTML, CSS ve SCSS kullanılarak geliştirilmiş olup, ikonlar için Font Awesome kullanılmıştır.

## 🛠️ Kullanılan Teknolojiler

* 🌐 HTML5
* 🎨 CSS3
* ⚡ SCSS (Sass)
* ⭐ Font Awesome

---

## ✨ Özellikler

* 📱 Tam responsive tasarım
* 🎯 Modern ve sade kullanıcı arayüzü
* ⚙️ SCSS ile modüler yapı
* 🔥 Font Awesome ikon desteği
* 🚀 Performans odaklı geliştirme
